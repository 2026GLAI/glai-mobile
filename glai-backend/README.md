# GLAi Backend v1

This repository contains the backend implementation of GLAi v1.

This codebase does not define behavior, rules, tone, or intent.

All authoritative definitions are located in:
https://github.com/2026GLAI/glai-mobile

This repository implements those documents exactly.

If behavior here diverges from glai-mobile,
it is considered a bug.

---

## Scope

GLAi Backend v1 is responsible for:

- session handling
- state transitions
- request validation
- safe response delivery
- boundary enforcement

It is not responsible for:

- product decisions
- UX logic
- tone optimization
- learning or adaptation

---

## Authority Rule

Documentation overrides code.

If code and documents disagree,
code must change.

---

## Version

This repository implements **GLAi v1 only**.
Any change requires explicit versioning.
